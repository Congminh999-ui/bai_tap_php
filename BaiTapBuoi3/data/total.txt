## LUỒNG HOẠT ĐỘNG TỔNG QUAN
```
1. User vào trang web upload.php
   ↓
2. PHP chạy dòng: require_once 'soluongnguoi.php'
   ↓
3. PHP nhảy sang file soluongnguoi.php:
   - Mở file online.txt
   - Đọc số cũ: 123
   - Tăng lên: 124
   - Ghi lại vào file: 124
   - Tạo biến $counter = 124
   ↓
4. PHP quay lại upload.php
   ↓
5. Hiển thị: "Lượt truy cập: 124"
   ↓
6. User thấy trang web + số lượt truy cập
